# ZF_Spider

这是一个用Python写的正方教务抢公选课脚本，由于每个学校的教务系统可能存在改动，所以不一定可用（DLU测试可用）。


使用方法：
config.json配置文件中填写教务系统的url，学号与密码，要抢的课程名称以及课程代码即可运行main.py

如果出现
“与*************
上课时间冲突！！”
字样，若之前该时间段无课的情况下，则证明选课成功，可停止脚本运行。
